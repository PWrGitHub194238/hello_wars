Hello Wars: DynaBlaster

Game overview

Hello Wars: DynaBlaster is a game resembling popular Bomberman. It consists of game board divided into small tiles. Each tile can be of different type:
  • Empty tile - self explanatory
  • Regular tile () - this is a tile that will get destroyed if a bomb explodes nearby
  • Fortified tile () - this is a tile that won't get destroyed by a bomb blast, but will turn into regular tile instead. So basically it is a tile with 2 HP
  • Indestructible tile () - this is a tile that won't get destroyed at all
